# Assets Directory

This directory contains all static assets for the website.

## Structure

### `/icons`
PWA icons and favicons in various sizes:
- favicon-16x16.png
- favicon-32x32.png
- apple-touch-icon.png (180x180)
- icon-72x72.png through icon-512x512.png

### `/images`
- `portrait.jpg` - Main hero portrait photo
- `og-image.jpg` - Open Graph image for social media (1200x630)
- Additional images as needed for sections

## Image Guidelines

### Portrait
- **Size:** 800x1000px (portrait orientation)
- **Format:** JPG or WebP
- **Style:** Warm lighting, professional, calm and confident expression
- **Background:** Minimal, soft gradient or blurred

### Open Graph Image
- **Size:** 1200x630px
- **Format:** JPG
- **Content:** Name, title, and subtle background
- **Text:** Large, readable on mobile

### Icons (PWA)
- **Format:** PNG with transparency
- **Sizes:** 72, 96, 128, 144, 152, 192, 384, 512 px
- **Design:** Simple "GK" monogram or Nexa-inspired symbol
- **Colors:** Electric blue (#2563EB) on dark background

## Optimization

All images should be:
- Compressed for web (TinyPNG, ImageOptim)
- Optimized for retina displays (2x resolution)
- Converted to WebP with fallbacks where supported
